using select command we can retrieve data from table in 3 different ways
    1) projection
    2)selection
    3)join

projection: to display only selected columns from the table.

mysql> select id,studentname from student;
+------+-------------+
| id   | studentname |
+------+-------------+
|  100 | tara        |
|  101 | arun        |
|  102 | pavan       |

selection:to display only selected row of data from the table.we use where clause to specify condition for selection.

mysql> select * from student
    -> where id=100 or id=104;
+------+-------------+----------------+---------+--------+
| id   | studentname | emailId        | address | mobile |
+------+-------------+----------------+---------+--------+
|  100 | tara        | tara@gmail.com | mandya  |   NULL |
|  104 | reny        | reny@gmail.com | mysore  |  12345 |
+------+-------------+----------------+---------+--------+
2 rows in set (0.05 sec)
-----------------------------------------------------------------
notes: selecet * from student;
      
       from-->    used to select table from db and put under the execution.
                  we can pass table name as an arguement.

       select-->  selects the required column field to prepare result set.
               
        *  ---->  selects all the columns of the table
                  we can also specify diff columns which are need to be displayed.


       order of execution : from , select
